# 🚀 AutoGravity Discord Bot

**AutoGravity** is an advanced all-in-one Discord management bot built to protect, manage, and grow your Discord server with AI-powered moderation, automation, welcome systems, auto roles, and slash commands.

![AutoGravity Banner](./assets/autogravity-banner.png)

---

## ✨ Features

### 🤖 AI Assistant
- Smart AI-powered community assistance
- Auto replies for common questions
- Helpful support for members and moderators

### 🛡️ Advanced Auto Moderation
- Toxic message detection
- Anti-spam protection
- Anti-scam and anti-phishing filters
- Link and invite filtering
- Auto-delete harmful content
- Anti-raid protection

### 👋 Welcome System
- Custom welcome messages
- Auto-DM onboarding
- Welcome and goodbye channels
- Verification workflow support

### 🎭 Auto Roles
- Automatic role assignment for new members
- Verification roles
- Level-based roles
- Reaction role support

### 🔨 Moderation Tools
- Ban, kick, mute, timeout, warn
- Temporary punishments
- Case tracking
- Moderator action logs
- Appeal system support

### ⚡ Slash Commands
- Fast and modern Discord commands
- Easy-to-use command system
- Fully customizable command structure

### 🎟️ Ticket System
- Support ticket creation
- Staff ticket management
- Ticket transcripts
- Auto-close and archive options

### 📊 Analytics & Reports
- Server statistics
- Member activity tracking
- Moderation reports
- Growth insights

### 🔒 Security Suite
- Account age verification
- Suspicious account detection
- Auto-lockdown mode
- Join-rate monitoring
- Role and permission protection

## 📦 Installation

```bash
git clone https://github.com/your-username/autogravity-discord-bot.git
cd autogravity-discord-bot
npm install
```

---

## ⚙️ Configuration

Create a `.env` file in the root directory:

```env
DISCORD_TOKEN=your_discord_bot_token
CLIENT_ID=your_discord_client_id
GUILD_ID=your_test_server_id
MONGO_URI=your_mongodb_connection_string
AI_API_KEY=your_ai_api_key
PREFIX=!
```

---

## ▶️ Run the Bot

```bash
npm start
```

For development:

```bash
npm run dev
```

## 💬 Example Commands

| Command | Description |
|---|---|
| `/help` | Show all available commands |
| `/ban` | Ban a member from the server |
| `/kick` | Kick a member from the server |
| `/timeout` | Timeout a member |
| `/warn` | Warn a member |
| `/clear` | Delete multiple messages |
| `/autorole` | Configure auto roles |
| `/welcome` | Configure welcome messages |
| `/ticket` | Create a support ticket |
| `/ai` | Ask the AI assistant |
